github账户申请和配置 登录git

没申请的赶紧申请吧，申请的用户名很重要

登录github后选择 new repository

repository的名字一定要和前面的名字一样，类似我的用户名是Hllele，后面新建名字是Hllele.github.io

使用github账号登录Git，打开电脑上开始菜单，找到Git，点击Git Bash

输入git config –global user.name “yourGitName”，回车

输入git config –global user.email “your email”，回车

hexo安装和配置

打开Git Bash，输入 npm install hexo-cli g

下载成功效果如下(网上找的图)

然后输入npm install hexo -save，然后就等

检测hexo是否安装成功，在Git Bash输入 hexo -v 出现类似以下就说明成功

记得要在Git Bash命令行输入，在win cmd下面输入会配置失败。

初始化hexo


在Git Bash命令行输入 npm install，npm会自动安装需要的组件。
